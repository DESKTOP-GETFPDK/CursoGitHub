Comandos de GitHub

--------------------------------------------------------------------------

git init: Comando para comenzar hacer un seguimiento a un proyecto. Este

crea dos areas: área de TRABAJO y área de ENSAYO(STAGIN AREA)

temporal o intermedia

--------------------------------------------------------------------------

git status -s: Comando que despliega el estatus de los archivos.

--------------------------------------------------------------------------

git add . :Comando para llevar los archivos del área de TRABAJO

al área de ENSAYO.

--------------------------------------------------------------------------

git commit -m "Nombre_de_Copia" : Comando para lleva los archivos del

área de ENSAYO al área de Repositorio con nombre de copia.
